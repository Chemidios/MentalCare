# MentalCare - Salud y Bienestar
Proyecto Final de Desarrollo de Interfaces.

## Funcionalidades
- **Diario Emocional:** Registro con emojis y notas personales.
- **Gestión de Hábitos:** Seguimiento diario persistente.
- **Seguridad Avanzada:** Bloqueo por PIN y Biometría (Huella/Cara).
- **Panel Admin:** Estadísticas globales y humor general.

## Tecnologías
- **Lenguaje:** Kotlin
- **UI:** Jetpack Compose (Material 3)
- **Persistencia:** Room Database
- **Arquitectura:** MVVM (Model-View-ViewModel)

## Instalación
1. Descarga el archivo `MentalCare_v1.0.apk`.
2. Habilita "Orígenes desconocidos" en tu móvil.
3. Instala y disfruta.

## Seguridad Avanzada: 
Implementación de Protected Routes mediante un interceptor de navegación que bloquea el acceso a los datos sensibles hasta que el usuario se autentica por PIN o Biometría.
